## PROJECT SUMMARY

**ONYX** is a high-end, mobile-first workout and nutrition orchestrator. It leverages Google Gemini AI to transform natural language "vibe inputs" into scientifically structured training and meal plans, delivered through a stealth-wealth, dark-mode interface.

---

## 2. TECHNICAL STACK

- **Framework:** Next.js 16 (App Router, Static Export ready)
- **UI:** React 19, TypeScript, Tailwind CSS v4
- **Components:** shadcn/ui (Radix Primitives)
- **Icons:** Lucide React (Stroke Width: 2)
- **Animations:** Framer Motion (Subtle spring physics & staggered entrances)
- **Data:** @tanstack/react-query (v5) for caching and optimistic updates
- **Auth/DB:** Supabase (Auth, Profiles, Daily Plans)
- **AI Engine:** Google Gemini (@google/genai) utilizing `gemini-2.5-flash-lite`
- **Persistence:** LocalStorage fallback for offline/demo reliability
